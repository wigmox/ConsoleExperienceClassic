## Quick Start

1. Download and extract to `Interface/AddOns/ConsoleExperienceClassic/`
2. Enable the addon at character select
3. Type `/ce` in-game to open configuration
4. See [Installation Guide](docs/Installation.md) for detailed instructions

### Steam Input (Recommended)

For the best controller experience:
1. Add WoW to Steam (name it "TurtleWoW")
2. Search for community config: **"WoW Vanilla Console experience by p3p0 v1.2"**
3. Adjust right joystick cursor settings for your screen resolution

## Compatibility

- **WoW Version**: Vanilla (1.12.1) / TurtleWoW
- **Dependencies**: None (standalone)
- **Optional**: [Interact.dll](https://github.com/luskanek/Interact) for enhanced interaction
